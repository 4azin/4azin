## 👋 감성을 더하는 개발자

기술은 편리함을 넘어, 사용자의 마음과 일상에 스며들 때 비로소 가치가 있다고 믿습니다.  
저는 **사람의 감정을 이해하고 반영하는 서비스**를 만들기 위해 고민하는 개발자입니다.

---

### 🌿 Background

- **생명과학 전공**으로 Bio 분야 전반에 대한 이해 보유  
  *(Bioinformatics / 생리학 / 신경학 / 면역학 / 계통진화학 등)*
- 콘텐츠 제작 경험을 통해 **스토리텔링과 감정 해석 능력** 습득
- **헬스케어 및 데이터 기반 서비스**에 관심

---

### 🛠 Tech & Role

- **Data & AI**: Python 기반 데이터 분석 및 머신러닝 학습  
  → *ADsP, SQLD 보유*
- **Backend**: Django, SQL  
- **Frontend**: JavaScript, Vue.js  
- **PM 경험** 기반의 문제 정의 및 서비스 구조 설계
- **생성형 AI 기반 UI/UX 설계** 경험

---

### 🎯 지향점

> **일상이 되는 서비스**  
사용자가 의식하지 않아도 자연스럽게 필요를 충족시키는 기술,  
그런 서비스를 만드는 것이 제 목표입니다.
